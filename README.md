# Daily Stock Report

매일 KST 08:00, 14:30에 관심종목 기술적 분석 + AI 가격 예측 + 숨겨진 종목 추천 리포트를 이메일로 발송합니다.

## 기능

- **관심종목 분석**: RSI, MACD, 볼린저 밴드, 이동평균선 기반 매매 신호
- **AI 가격 예측**: Kronos-mini Foundation Model로 1일/5일 확률적 가격 예측 (예상 범위, 상승 확률, 변동성)
- **숨겨진 종목 추천**: KOSPI/KOSDAQ 중 거래량 급증 + RSI 과매도 종목 Top 10
- **HTML 리포트**: 깔끔한 테이블 형식으로 Gmail 발송
- **멀티마켓**: 한국(KRX), 미국(US) 주식 동시 지원 (57종목)

## 종목 관리

`watchlist.json`을 편집하면 됩니다:

```json
{
  "stocks": [
    {"name": "삼성전자", "ticker": "005930.KS", "market": "KRX"},
    {"name": "애플", "ticker": "AAPL", "market": "US"}
  ]
}
```

### 티커 형식

| 시장 | 형식 | 예시 |
|---|---|---|
| KOSPI | 종목코드.KS | 005930.KS |
| KOSDAQ | 종목코드.KQ | 035900.KQ |
| US | 심볼 | AAPL, MSFT |

## 아키텍처

```
GitHub Actions (cron: 매일 KST 08:00, 14:30) — timeout 20분
  └─ ubuntu-latest + Python 3.12 + PyTorch CPU
       ├─ market_overview.py  → 시장 요약 (지수, 환율, VIX, 섹터)
       ├─ sort_utils.py       → 시장/시총 기반 워치리스트 정렬
       ├─ analyzer.py         → 관심종목 기술적 분석 (RSI, MACD, BB, SMA)
       ├─ ai_forecast.py      → AI 가격 예측 (Kronos-mini, Monte Carlo 5회)
       ├─ extras.py           → 52주 알림, 주간 성과, 실적 캘린더
       ├─ gem_scanner.py      → 숨겨진 종목 추천 (KOSPI/KOSDAQ 70종목 스캔)
       ├─ report_generator.py → HTML 리포트 조합
       └─ send_email.py       → Gmail SMTP 발송 (다중 수신인)

  libs/
       └─ kronos_predictor.py → Kronos Foundation Model 래퍼
```

## GitHub Secrets 설정

| Secret | 값 |
|---|---|
| `GMAIL_USER` | 발신 Gmail 주소 |
| `GMAIL_APP_PASSWORD` | Gmail 앱 비밀번호 (16자리) |
| `REPORT_EMAIL` | 수신 이메일 주소 |

## 수동 실행

Actions 탭 → Daily Stock Report → Run workflow

## 로컬 테스트

```bash
pip install -r requirements.txt
cd scripts
python report_generator.py ../watchlist.json > report.html
open report.html
```
